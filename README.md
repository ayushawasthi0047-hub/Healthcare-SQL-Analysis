# Healthcare SQL Analysis

## Project Overview
SQL analysis of 500,000 Indian hospital patient records to uncover 
treatment cost patterns, readmission rates, and hospital performance insights.

## Dataset
- 500,000 patient records (2019-2024)
- 18 columns: Patient demographics, medical conditions, 
  hospital details, treatment costs, readmission status
- Indian hospitals, cities and states

## Tools Used
- SQL Server (SSMS)
- SQL: GROUP BY, JOINs, CTEs, Window Functions, 
  CASE WHEN, Subqueries, CAST, ROUND

## Business Questions Answered
1. Which medical condition has highest average treatment cost?
2. Which hospital generates maximum revenue?
3. Which condition has highest patient readmission rate?
4. What is the year-wise patient admission trend?
5. Which states have highest patient load?
6. Which insurance provider covers most patients?
7. What medications are most prescribed per condition?

## Key Insights
- Cancer has the highest average treatment cost at ₹2,90,000
- Heart Disease patients show highest readmission rate at 20%
- AIIMS Delhi and Apollo Hospital are top revenue generating hospitals
- Patient admissions grew consistently from 2019 to 2024
- Maharashtra and Karnataka have highest patient volumes
- Self-Pay patients incur 35% higher costs than insured patients

## SQL Concepts Used
| Level | Concepts |
|-------|----------|
| Basic | SELECT, WHERE, GROUP BY, ORDER BY, COUNT, SUM |
| Intermediate | AVG, ROUND, CAST, CASE WHEN, HAVING |
| Advanced | CTEs, Window Functions (RANK, SUM OVER), Subqueries |
